# UI/UX Design Brief

## Product feeling
Professional internal audit system, not a flashy dashboard.

## Key pages
1. Login
2. Dashboard
3. Report Periods
4. Create Report
5. Upload Excel Files
6. Validation Workspace
7. Inventory Dashboard
8. Report Builder
9. Historical Reports
10. Settings

## Login page
- App title
- Company logo placeholder
- Sign in with Google button
- Clean, minimal layout

## Validation workspace
This is the most important screen.
Show warnings clearly before dashboard generation.

## Empty states
Never show fake sample data. If no report data exists, show a clear empty state.

## Zero-cost storage warning
On report generation screen, tell user:
"Generated PDFs are downloaded locally in the zero-cost version. They are not stored in cloud storage."
