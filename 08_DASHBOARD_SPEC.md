# Dashboard Specification

## Dashboard purpose
Explain inventory accuracy, variance, issue reasons, and pending action. Do not only show decorative charts.

## KPI cards
- Total items counted
- Matched items
- Mismatched items
- Match %
- Shortage items
- Excess items
- Gross variance value
- Pending action items

## Charts/tables
- Matched vs mismatched
- Shortage vs excess
- Issue category breakdown
- Supplier-wise variance
- Open vs closed status
- Top high-risk items
- Pending action table

## Filters
- Report period
- Supplier
- Issue category
- Status
- Matched/Shortage/Excess
- High variance

## Rule
Dashboard must be locked until validation is approved.
